# 🔐 Password Analyzer & Generator

A simple web tool that checks password strength and generates secure passwords.

---

## 🚀 Features

- Check password strength (Weak / Medium / Strong)
- Generate strong passwords automatically
- Copy password with one click
- Password suggestions
- Show/Hide password option

---

## 🛠️ Built With

- HTML
- CSS
- JavaScript

---

## 📂 Files

- index.html → Main UI + code
- style.css → Design (inside file or external if separated)
- script.js → Password logic

---

## ▶️ How to Use

1. Open `index.html` in browser
2. Type any password to check strength
3. Click “Generate” to create strong password
4. Copy and use it
