# 🔐 Password Analyzer & Generator

A simple web tool that checks password strength and generates secure passwords.

---

## 🚀 Features

- Password strength checker (Weak / Medium / Strong)
- Strong password generator
- Copy suggestion / Generate password 
- Smart suggestions
- Show/Hide password
